Analista Programador Sr. Java


Información

Argentina

Buenos Aires

Tiempo Completo

Importante empresa de Telecomunicaciones se encuentra en la búsqueda de un Analista Programador Java SSr o Sr, para incorporarse al equipo de trabajo de desarrollo de software.

Nos orientamos a una persona con iniciativa y creatividad para participar en el desarrollo de soluciones tecnológicas para nuestros clientes, quienes confían en nosotros la tarea de facilitarle sus operaciones. Se trabajará en equipo, por lo que es importante que quien ingrese pueda generar buenas relaciones interpersonales y posea espíritu colaborativo.

Para poder llevar adelante las tareas, es sumamente importante poder analizar completamente los contextos del desarrollo, para poder anticiparse, asesorar y facilitar la comprensión de las diferentes eventualidades y soluciones que se propongan.

Requisitos

  • Estudiante o graduado en carreras de informática.
  • Experiencia en programación JAVA y J2EE mínimo de 2 años, con utilización de Oracle WebLogic, SOA Suite, MQ, JMS o similares.
  • Conocimientos de herramientas de monitoreo.
  • Tener experiencia en arquitectura de redes y arquitectura de software en entornos Linux / UNIX / Windows.
  • Inglés avanzado.
